# ☕ Starbucks Sales Analysis Dashboard | Power BI

## Project Overview

This Power BI project provides an interactive analysis of Starbucks sales performance, customer purchasing behavior, and product category trends. The dashboard helps stakeholders monitor key business metrics and make data-driven decisions.

## Objectives

* Analyze overall sales performance.
* Track order quantity and revenue trends.
* Identify top-performing products and categories.
* Understand customer purchasing patterns.
* Compare sales across different store locations.

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization

## Key Performance Indicators (KPIs)

* Total Sales
* Total Orders
* Total Quantity Sold
* Average Order Value
* Sales by Product Category
* Sales by Store Location
* Monthly Sales Trends

## Dashboard Features

### Executive Summary

* Overall business performance overview
* Key sales metrics
* Revenue insights

### Product Analysis

* Best-selling products
* Category-wise sales distribution
* Product performance comparison

### Store Performance

* Location-wise sales analysis
* Revenue contribution by store
* Order volume comparison

### Time-Based Analysis

* Daily sales trends
* Monthly performance tracking
* Peak sales periods

## Business Insights

* Identifies top revenue-generating products.
* Highlights high-performing store locations.
* Tracks customer demand across product categories.
* Enables data-driven inventory and sales planning.
